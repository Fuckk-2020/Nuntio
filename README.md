Nuntio 🌀

A minimal pomodoro desktop app. *tempus fugit.*

 Features
-  25/5 Pomodoro timer
-  Task list
-  Draggable sticky notes
-  Movable window
  

## How to use
1. Install [Node.js](https://nodejs.org)
2. Clone this repo
3. Run in terminal:
```
npm install
npm run build
.\node_modules\.bin\electron .
```

Or just double click `Nuntio.bat`
